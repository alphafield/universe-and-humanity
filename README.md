# Universe and Humanity（宇宙与人类）

A minimalist, structural interface between the Universe and Humanity.

This repository is **not** a belief system and not a philosophical doctrine.
It is an attempt to state **what must hold** for complex systems to persist — across physics, life, and civilization.

## Live
- Website: https://universeandhumanity.com/

## What’s inside
- `index.html` — landing page (Hero + scroll + scientists platform)
- `styles.css` — styles (dark, minimal, SpaceX-like whitespace)
- `app.js` — Three.js galaxy background + interactions
